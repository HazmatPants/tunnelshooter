# How to export

1. Open the project in the editor

![alt text](/repo-assets/how2export/1.png)

2. Go to Project > Export

![alt text](/repo-assets/how2export/2.png)

3. Click "Add..." and choose your OS (there may be some for Windows and Linux already, you can select one of those)

![alt text](/repo-assets/how2export/3.png)

You may get errors like this:

![alt text](/repo-assets/how2export/4.png)

Click "Manage Export Templates"
Then "Download and Install" and wait for it to download.

![alt text](/repo-assets/how2export/5.png)


4. Now click "Export Project" and choose where to export it to

![alt text](/repo-assets/how2export/6.png)

5. Now the game is ready to play, just run the executable.
`tunnelshooter.x86_64` or `tunnelshooter.sh` on Linux, `tunnelshooter.exe` on Windows
